## 1. 产品背景

**一句话**：经典斗地主 × Roguelike 肉鸽构筑的单机策略卡牌游戏。

- 售价 $4.99 / ¥25，买断制无广告
- 目标用户：20-40 岁，喜欢策略卡牌的玩家
- 核心卖点：60+ 种规则牌（Joker）、9 种流派、15 层挑战
- App Store 分类：Games → Card / Strategy

---

## 2. 交付规格

| 项目 | 要求 |
|------|------|
| **尺寸** | 1024 × 1024 px |
| **格式** | PNG，无透明度（no alpha） |
| **色彩空间** | sRGB |
| **圆角** | ⚠️ 不要手动加圆角，iOS 系统自动裁剪（约 20% 半径） |
| **安全区域** | 四角向内 ~15% 不放关键元素（会被圆角遮挡） |
| **最小辨识** | 缩小到 60×60 px 仍需清晰辨识 |

---

## 3. 品牌色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#1A0F08` → `#0D0704` | 深棕/近黑渐变 |
| 主金色 | `#D9AD47` | 文字、图标主色 |
| 高光金 | `#EBC85F` | 光效、强调 |
| 暗金 | `#8B6914` | 阴影、边框 |
| 点缀红 | `#C23030`（可选） | 中国红，少量使用 |

**总体感觉**：暗色奢华 + 金色高级感，类似高端卡牌游戏的质感。

---

## 4. 必含设计元素

### ♠ 黑桃符号（核心视觉焦点）
- 经典扑克♠造型，**金色**填充
- 位于画面中心偏上，作为视觉主焦点
- 尖顶、两侧饱满叶瓣、窄腰、短茎——标准扑克黑桃

### 🀄 "斗" 字
- 位于黑桃下方或画面下部
- 书法感/毛笔风格优先，也可用粗衬线字体
- 金色，带微妙发光或阴影
- ⚠️ 不要放太靠底部，会被 iOS 圆角裁剪

### 🃏 扑克牌暗示（可选）
- 卡牌边框、A 牌角标、牌面纹理等
- 作为背景层或装饰元素，不要喧宾夺主

### ✨ 氛围元素（可选）
- 金色粒子/光点散落
- 微妙的放射光效
- 中式边角纹饰（极简）

---

## 5. 不要出现的元素

| ❌ 避免 | 原因 |
|---------|------|
| 过多文字 | "斗" 一个字足够，不要放完整游戏名 |
| 写实人物 | 不符合产品风格，且缩小后模糊 |
| 鲜艳多色 | 保持深色+金色双色调为主 |
| 3D 渲染感 | 偏平面/2.5D 质感，避免廉价 3D |
| 赌博暗示 | 不要出现骰子、筹码、赌桌——年龄分级 4+ |
| 手动圆角 | iOS 自动处理 |

### 值得参考的图标风格
| 游戏 | 参考点 |
|------|--------|
| **Balatro** | 小丑牌单焦点 + 暗色背景，高辨识度 |
