# 斗破乾坤 — 商业化战略文档 (Monetization Strategy)

> 版本: v1.0 | 更新日期: 2026-04-24

## 1. Executive Summary

**斗破乾坤**是一款单机 Roguelike 斗地主卡牌游戏，定位"Balatro × 中国斗地主"。核心商业模式采用 **免费试玩 + 买断制 + DLC 扩展** 三层递进策略。

**第一年保守营收目标: ¥500,000+**

三大收入支柱:
1. **核心买断**: ¥18-¥28 完整版解锁 (贡献约60%收入)
2. **DLC 内容包**: ¥6-¥12/个 (贡献约25%收入)
3. **Steam 移植**: $4.99 跨平台收入 (贡献约15%收入)

## 2. Market Analysis (市场分析)

### 2.1 市场规模
- 中国手游卡牌品类 2025 年市场规模约 ¥380 亿
- 斗地主用户月活约 1.5 亿, 是中国最大扑克品类
- Roguelike 品类增速 40%/年, 用户付费意愿高于均值

### 2.2 竞品营收基准

| 竞品 | 模式 | 定价 | 预估营收 |

| Balatro | 买断制 | $14.99 | $50M+ (2024年) |

| 杀戮尖塔 Mobile | 买断制 | ¥28 | $10M+ |
| 斗地主(腾讯) | F2P+广告 | - | ¥10B+/年 |

### 2.3 我们的定位
- **非竞技**: 不与腾讯斗地主竞争, 做"斗地主+Roguelike"细分蓝海
- **Premium Casual**: 无广告, 无体力, 买断即拥有
- **目标用户**: 25-40岁, 会玩斗地主, 喜欢策略卡牌, 愿为品质付费

**定价策略**:

## 4. Conversion Funnel (转化漏斗设计)

```
下载 (100%)
  ↓ 首次打开
教程完成 (85%)   ← 流畅的8步教程
  ↓ 选择流派开始
完成Floor 1-2 (70%)  ← 核心玩法钩子
  ↓ 进入商店购买规则牌
完成Floor 3商店 (60%)  ← 感受规则牌的力量
  ↓ 挑战第一个Boss
Floor 4 Boss战 (50%)  ← 策略深度展示
  ↓ DemoGate付费墙
查看付费页 (45%)
  ↓ 决定购买
转化购买 (6-8%)  ← 目标转化率
```

**关键优化点**:
1. Floor 3 商店体验: 确保玩家能买到明显增强体验的规则牌
2. Floor 4 Boss: 展示Boss修改器带来的策略变化
3. DemoGate页面: 展示剩余11个关卡+Ascension+收藏系统
4. 价格锚定: 标注原价¥28, 限时优惠¥12

## 5. Retention Mechanics (留存系统)

### 5.1 每日挑战 → D1/D7 留存
- 每天一个固定种子的挑战关
- 全球玩家同一关卡 → 社交讨论
- 完成奖励: 50金币 + 成就进度
- **目标**: D1留存从30%→45%, D7从15%→25%

### 5.2 Ascension 阶梯 → D30 留存
- 10级渐进难度, 每级增加新规则
- 通关A10为终极成就
- **目标**: D30留存从8%→15%

### 5.3 收藏/图鉴系统 → 长期目标
- "发现所有规则牌效果": 驱动重复游玩
- "解锁所有成就": 17+成就目标
- "尝试所有流派": 9种起始流派

### 5.4 周排行榜 (Phase 2)
- 每周总分排名
- 好友排行榜 (Game Center)
- 奖励: 排名前10%获得专属牌背
