## 二、竞品基准与定位

### 市场验证
- **Balatro**: 买断$14.99，400万+销量，$50M+收入 → 证明扑克Roguelike天花板极高
- **月圆之夜**: 免费+DLC模式，中国市场¥198M+累计 → 证明中国市场DLC模式可行
- **杀戮尖塔Mobile**: 买断¥28，$10M+ → 证明Roguelike卡牌手游有稳定付费用户
- **斗地主(腾讯)**: 月活1.5亿 → 证明斗地主品类用户基数庞大

### 差异化定位
```
细分蓝海 = 斗地主手型 × Roguelike构筑 × 单机买断
竞争对手: 零 (全球无直接竞品)
```
