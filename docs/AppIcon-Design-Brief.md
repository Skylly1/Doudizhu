# 🎨 App Icon 优化设计 Brief — 斗破乾坤

> 日期：2026-04-27 | 优先级：P0（上架必须）

---

## 问题

当前 App Icon 存在两个核心问题：
1. **视觉精致度不够** — 使用 SF Symbol 系统图标（黑桃♠），缺乏手绘质感和品牌辨识度
2. **与 APP 首页风格脱节** — Icon 用冷调墨紫背景，首页用暖棕国潮渐变，视觉语言不统一

## 目标

设计一版与 APP 首页视觉风格完全一致的 App Icon，一眼传达「国潮 × 卡牌策略」的品牌调性。

---

## 当前 Icon 风格 vs APP 首页风格

| 对比维度 | 当前 Icon ❌ | APP 首页 ✅（目标方向） |
|---------|------------|-------------------|
| 背景色调 | 冷调墨紫黑 `rgb(20,15,31)` | 暖棕渐变 `rgb(97,66,41)` → `rgb(43,31,23)` |
| 主体元素 | SF Symbol 黑桃 + 文字堆叠 | 菱形印章「斗」字 + 金色渐变辉光 |
| 金色表现 | 简单线性渐变 | 多层辉光 + 径向光晕 + 呼吸感 |
| 质感层次 | 纯色平面 | 粒子光点 + 烟雾氛围 + 宣纸纹理 |
| 整体气质 | 偏暗偏冷，像工具类 App | 温暖大气，国潮游戏质感 |

## 设计方向

### 整体构图

居中构图，菱形印章「斗」字作为绝对视觉主体，占画面约 55~60%。

```
┌──────────────────────┐
│                      │
│    ◇ 金色辉光圈 ◇     │
│      ╱╲              │
│     ╱  ╲  ← 外菱形框  │
│    ╱ 斗 ╲             │
│    ╲    ╱  ← 内菱形框  │
│     ╲  ╱             │
│      ╲╱              │
│   · · · · ← 金色微粒   │
│                      │
└──────────────────────┘
```

### 1. 背景层

- **主渐变**：从左上到右下，暖棕色
  - 起始：`#614229`（rgb 97, 66, 41）
  - 中间：`#422E1E`（rgb 66, 46, 30）
  - 终止：`#2B1F17`（rgb 43, 31, 23）
- **叠加**：居中金色径向光晕
  - 颜色：`rgba(230, 184, 77, 0.20)`
  - 半径：覆盖画面 60% 范围
- **纹理**：微妙宣纸肌理，5~8% 透明度叠加（增加质感但不抢主体）

### 2. 主体层 — 菱形印章「斗」

复用首页的印章设计语言：

- **外菱形框**：圆角矩形旋转 45°，金色描边
  - 描边色：`#D4A338`（rgb 212, 163, 56），描边宽度约 2.5%
  - 透明度：55%
- **内菱形框**：略小一圈，更细描边
  - 描边色：`#D4A338`，透明度 22%
- **「斗」字**：
  - 字体：衬线体（思源宋体/方正清刻本悦宋 或等效中文书法衬线）
  - 大小：占印章内框约 75%
  - 渐变填充（从上到下）：
    - `#F2D16B`（rgb 242, 209, 107）
    - `#D4A338`（rgb 212, 163, 56）
    - `#A67A1F`（rgb 166, 122, 31）
  - 投影：黑色 50% 透明度，轻微下偏移

### 3. 氛围层

- **辉光环**：印章外围柔和金色光晕
  - 颜色：`rgba(212, 163, 56, 0.35)`
  - 模糊半径：约画面 15%
- **金色微粒**：3~5 个大小不一的光点散布在下半部
  - 大颗粒：直径约画面 0.8%，带辉光
  - 小颗粒：直径约画面 0.4%，无辉光
  - 颜色：`#D9AD47`

## 交付要求

| 项目 | 规格 |
|------|------|
| 尺寸 | **1024 × 1024 px** |
| 格式 | PNG |
| 透明度 | **无透明**（必须填满背景） |
| 圆角 | **不加圆角**（iOS 系统自动裁切） |
| 色彩空间 | sRGB |
| 附加 | 额外提供一张 iOS 圆角遮罩预览图用于确认效果 |

---

## 避免事项

- ❌ 不要使用 SF Symbol 或系统图标
- ❌ 不要用冷色调（紫/蓝/灰）作为背景
- ❌ 不要放过多文字（只保留「斗」字）
- ❌ 不要有过于复杂的装饰元素（App Icon 小尺寸看不清）
- ❌ 不要有透明区域或圆角
